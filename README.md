# react-admin

react + redux + react-router + sass + antd + eslint + webpack 后台管理系统Demo

# 使用方法

先执行以下命令，dll化常用的第三方模块，只需dll一次，生成对应文件即可，后续这些第三方模块没更新的情况下，都不需要再次dll了。dll化可以提高webpack构建速度，也有利于客户端缓存。

```shell
npm run dll
```

执行以下命令，可进行开发

```shell
npm run dev
```

执行以下命令，可打包项目

```shell
npm run build
```
